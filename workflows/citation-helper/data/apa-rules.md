# APA Citation Rules

**APA (American Psychological Association) - 7th Edition**

---

## General APA Formatting Principles

1. **Author Names:**
   - Lastname, First Initial. Middle Initial.
   - Example: Smith, J. A.

2. **Title Formatting:**
   - Sentence case (only first word and proper nouns capitalized)
   - Italicize book titles and journal names
   - Article titles not italicized

3. **Publication Year:**
   - In parentheses immediately after author
   - Example: (2020)

4. **Order of Elements:**
   - Author (year). Title. Publication information.

---

## Source Type Templates

### Book

**Format:**
```
Author, A. A. (Year). Title of book: Subtitle in sentence case. Publisher.
```

**Example:**
```
Silva, J. C. (2020). Introduction to scientific research: A practical guide. Atlas Press.
```

**Required fields:**
- Author (Lastname, Initials)
- Year
- Title (italicized, sentence case)
- Publisher

**Optional fields:**
- Edition (in parentheses after title)
- DOI or URL (if e-book)

---

### Journal Article

**Format:**
```
Author, A. A., & Author, B. B. (Year). Title of article in sentence case. Journal Name in Title Case, volume(issue), page-page. https://doi.org/xxx
```

**Example:**
```
Oliveira, M. (2020). The importance of proper citation in academic writing. Brazilian Journal of Education, 25(3), 45-62. https://doi.org/10.1234/bje.2020.0001
```

**Required fields:**
- Author (Lastname, Initials)
- Year
- Article title (sentence case, not italicized)
- Journal name (italicized, title case)
- Volume number (italicized)
- Issue number (in parentheses, not italicized)
- Page range
- DOI (if available)

**If no DOI:**
- Omit DOI, end with page numbers
- OR include URL if freely available online

---

### Website / Online Document

**Format:**
```
Author, A. A., or Organization Name. (Year, Month Day). Title of page in sentence case. Website Name. URL
```

**Example:**
```
University of São Paulo. (2021, January 15). Guidelines for academic writing. USP Portal. https://www.usp.br/guidelines
```

**Required fields:**
- Author or organization
- Date (Year, Month Day) or (n.d.) if no date
- Title (sentence case)
- Website name (if different from author)
- URL

**Note:** No retrieval date needed unless content changes frequently

---

### Thesis / Dissertation

**Format:**
```
Author, A. A. (Year). Title of thesis in sentence case [Doctoral dissertation or Master's thesis, Institution Name]. Database or Archive Name. URL
```

**Example:**
```
Santos, A. P. (2019). Human resources management: A case study [Master's thesis, University of São Paulo]. USP Digital Library. https://www.teses.usp.br/xxxxx
```

**Required fields:**
- Author
- Year
- Title (italicized, sentence case)
- Degree type [in brackets]
- Institution
- Database or repository name
- URL (if available online)

---

### Conference Paper (Proceedings)

**Format:**
```
Author, A. A. (Year, Month Day-Day). Title of paper in sentence case [Conference presentation or Paper presentation]. Conference Name, Location. URL
```

**Example:**
```
Costa, P. (2020, June 10-12). Innovation in Brazilian SMEs [Paper presentation]. 5th Symposium on Management and Innovation, Rio de Janeiro, Brazil.
```

---

### E-book

**Format:**
```
Author, A. A. (Year). Title of book in sentence case. Publisher. https://doi.org/xxx
```

**Example:**
```
Ferreira, L. (2021). Digital marketing strategies (2nd ed.). Saraiva Publishing. https://doi.org/10.1234/ebook.2021
```

**Note:** Format is same as print book, but include DOI or URL

---

### Newspaper Article

**Format:**
```
Author, A. A. (Year, Month Day). Title of article in sentence case. Newspaper Name in Title Case. URL
```

**Example:**
```
Almeida, R. (2022, January 20). Brazilian economy shows signs of recovery. Folha de S.Paulo. https://www.folha.uol.com.br/article-url
```

---

### Government Document / Report

**Format:**
```
Government Agency. (Year). Title of report in sentence case (Report No. xxx). Publisher. URL
```

**Example:**
```
Ministry of Education. (2014). National education plan 2014-2024 (Report No. MEC-2014-001). Brazilian Government Press. https://www.gov.br/report-url
```

## In-Text Citations (APA 7th Edition)

### Parenthetical Citations

**Paraphrase (one author):**
```
(Smith, 2020)
```

**Paraphrase (two authors):**
```
(Smith & Johnson, 2020)
```

**Paraphrase (three or more authors):**
```
(Smith et al., 2020)
```

**Direct quote:**
```
(Smith, 2020, p. 45)
or for multiple pages:
(Smith, 2020, pp. 45-47)
```

### Narrative Citations

**One author:**
```
Smith (2020) found that...
```

**Two authors:**
```
Smith and Johnson (2020) argued that...
```

**Three or more authors:**
```
Smith et al. (2020) demonstrated that...
```

**Direct quote:**
```
As Smith (2020) stated, "quote text here" (p. 45).
```

### Special Cases

**Same author, different years:**
```
(Smith, 2018, 2020)
```

**Same author, same year:**
```
(Smith, 2020a, 2020b)
```

**Multiple sources:**
```
(Johnson, 2019; Smith, 2020; Williams, 2021)
```

**No author:**
```
("Article Title," 2020)
```

**No date:**
```
(Author, n.d.)
```

**Organization as author (first citation):**
```
(American Psychological Association [APA], 2020)
```

**(Subsequent citations):**
```
(APA, 2020)
```

## Key APA Rules Summary

1. ✅ Author: Lastname, F. M. (initials only)
2. ✅ Year in parentheses after author
3. ✅ Titles in sentence case
4. ✅ Italicize book titles and journal names
5. ✅ DOI preferred over URL (when available)
6. ✅ Hanging indent for reference list entries
7. ✅ Alphabetical order by author surname
8. ✅ "et al." for 3+ authors in citations
9. ✅ Use "&" in parenthetical citations, "and" in narrative citations
10. ✅ No retrieval date for websites (unless content changes frequently)

## Common DOI Formats

**Preferred format:**
```
https://doi.org/10.1234/example.2020.001
```

**NOT:**
```
doi: 10.1234/example.2020.001
DOI: 10.1234/example.2020.001
dx.doi.org/10.1234/example.2020.001
```

## Capitalization Rules (Sentence Case)

**Correct:**
```
Introduction to scientific research: A practical guide
The impact of social media on education
```

**Incorrect:**
```
Introduction To Scientific Research: A Practical Guide
The Impact Of Social Media On Education
```

**Exception:** Proper nouns are always capitalized
```
Brazilian economy in the 21st century
COVID-19 and education policy
```
