# ABNT Citation Rules

**ABNT (Associação Brasileira de Normas Técnicas) - Brazilian Citation Standards**

**Primary Standards:**
- ABNT NBR 10520:2023 - In-text citations
- ABNT NBR 6023:2018 - References (most common)
- ABNT NBR 6023:2025 - References (latest version)

---

## General ABNT Formatting Principles

1. **Author Names:**
   - Surname in UPPERCASE (CAPS)
   - Followed by first name or initials
   - Example: SILVA, João Carlos.

2. **Title Formatting:**
   - **Article/chapter titles:** Regular text
   - **Book titles:** Bold
   - **Journal/publication names:** Bold

3. **Punctuation:**
   - Period after each element
   - Comma between author name parts

4. **Order of Elements:**
   - Author → Title → Publication details → Year

## Source Type Templates

### Book

**Format:**
```
AUTHOR SURNAME, First name(s). **Book title: subtitle**. Edition (if not first). City: Publisher, Year.
```

**Example:**
```
SILVA, João Carlos. **Introdução à pesquisa científica**. 3. ed. São Paulo: Atlas, 2020.
```

**Required fields:**
- Author (surname in CAPS)
- Title (bold)
- Edition (if not first: 2. ed., 3. ed.)
- City (publisher location)
- Publisher
- Year

**Optional fields:**
- Translator
- Pages (total pages: 250 p.)
- ISBN

---

### Journal Article

**Format:**
```
AUTHOR SURNAME, First name(s). Title of article. **Journal Name**, v. volume, n. issue, p. start-end, Month Year.
```

**Example:**
```
OLIVEIRA, Maria. A importância da citação. **Revista Brasileira de Educação**, v. 25, n. 3, p. 45-62, set. 2020.
```

**Required fields:**
- Author (surname in CAPS)
- Article title (regular text)
- Journal name (bold)
- Volume (v. X)
- Issue number (n. Y)
- Pages (p. XX-YY)
- Month (abbreviated: jan., fev., mar., abr., mai., jun., jul., ago., set., out., nov., dez.)
- Year

**Optional fields:**
- DOI: Available at: https://doi.org/xxx. Accessed on: dd Month yyyy.
- ISSN

---

### Website / Online Document

**Format:**
```
AUTHOR SURNAME, First name(s) (or ORGANIZATION). Title of page/document. **Website Name**, Year. Available at: URL. Accessed on: dd Month yyyy.
```

**Example:**
```
UNIVERSIDADE DE SÃO PAULO. Normas para apresentação de trabalhos. **Portal USP**, 2021. Available at: https://www.usp.br/normas. Accessed on: 15 jan. 2022.
```

**Required fields:**
- Author or organization (in CAPS)
- Title
- Website name (bold) or description
- Year (or access year if no publication year)
- URL
- Access date

---

### Thesis / Dissertation

**Format:**
```
AUTHOR SURNAME, First name(s). **Thesis title: subtitle**. Year. Number of pages. Degree (Master's/Doctoral) – Institution, City, Year.
```

**Example:**
```
SANTOS, Ana Paula. **Gestão de recursos humanos: estudo de caso**. 2019. 150 f. Dissertation (Master's in Business Administration) – Faculdade de Economia e Administração, Universidade de São Paulo, São Paulo, 2019.
```

**Required fields:**
- Author (surname in CAPS)
- Title (bold)
- Year
- Number of pages (f. = folhas/sheets)
- Degree type and area
- Institution
- City
- Year of defense

---

### Conference Paper

**Format:**
```
AUTHOR SURNAME, First name(s). Title of paper. In: NAME OF CONFERENCE, edition number., year, location. **Proceedings...**  City: Publisher, Year. p. start-end.
```

**Example:**
```
COSTA, Pedro. Inovação em PMEs brasileiras. In: SIMPÓSIO DE GESTÃO E INOVAÇÃO, 5., 2020, Rio de Janeiro. **Anais...** Rio de Janeiro: UFRJ, 2020. p. 120-135.
```

---

### E-book

**Format:**
```
AUTHOR SURNAME, First name(s). **Book title: subtitle**. Edition. City: Publisher, Year. E-book. Available at: URL or platform. Accessed on: dd Month yyyy.
```

**Example:**
```
FERREIRA, Lucas. **Marketing digital**. 2. ed. São Paulo: Saraiva, 2021. E-book. Available at: https://plataforma.saraiva.com.br. Accessed on: 10 mar. 2022.
```

---

### Newspaper Article

**Format:**
```
AUTHOR SURNAME, First name(s). Title of article. **Newspaper Name**, City, dd Month yyyy. Section, p. page.
```

**Example:**
```
ALMEIDA, Roberto. Economia brasileira em recuperação. **Folha de S.Paulo**, São Paulo, 20 jan. 2022. Economia, p. A12.
```

---

### Government Document / Report

**Format:**
```
COUNTRY or ORGANIZATION. Ministry/Department. **Document title**. City, Year. Pages.
```

**Example:**
```
BRASIL. Ministério da Educação. **Plano Nacional de Educação 2014-2024**. Brasília, 2014. 86 p.
```

## In-Text Citations (ABNT NBR 10520:2023)

### Author-Date System (Most Common)

**Paraphrase:**
```
According to Silva (2020), the methodology...
or
The methodology described (SILVA, 2020) shows...
```

**Direct Quote (Short - less than 3 lines):**
```
"Citation text here" (SILVA, 2020, p. 45).
```

**Direct Quote (Long - 3+ lines):**
```
Indent 4cm from left margin, size 10 font, single spacing:
    Long quoted text here spanning multiple lines
    continues without quotation marks.
(SILVA, 2020, p. 45-46).
```

**Multiple Authors:**
- 2 authors: Silva and Costa (2020) or (SILVA; COSTA, 2020)
- 3+ authors: Silva et al. (2020) or (SILVA et al., 2020)

**Same Author, Same Year:**
- Add letters: Silva (2020a), Silva (2020b)

**Multiple Sources:**
- Chronological order: (SILVA, 2018; COSTA, 2019; OLIVEIRA, 2021)

## Common Abbreviations

- **ed.** = edição (edition)
- **v.** = volume (volume)
- **n.** = número (number/issue)
- **p.** = página(s) (page/pages)
- **f.** = folhas (sheets - for theses)
- **et al.** = et alii (and others)
- **org.** = organizador (editor/organizer)
- **coord.** = coordenador (coordinator)
- **comp.** = compilador (compiler)

## Key ABNT Rules Summary

1. ✅ Author surname in CAPS
2. ✅ **Bold** for book titles and publication names
3. ✅ Regular text for article/chapter titles
4. ✅ Alphabetical order by author surname
5. ✅ Period after each element
6. ✅ Consistent formatting throughout bibliography
7. ✅ Include access date for online sources
8. ✅ Use standard abbreviations (v., n., p., ed.)
