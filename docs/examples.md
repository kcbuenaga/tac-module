## Example Workflow: First Time User

**Scenario:** Maria is an MBA student at USP with no thesis topic, procrastinating for three weeks.

**Journey:**

1. Installs TAC and configures Portuguese language
2. Runs Library Integration Setup with Patricia
3. Uses Topic Discovery & Validation with Dr. Carla
4. Gets validated topic with 34 sources
5. **Breakthrough moment** — no longer frozen!
